Enum ConsoleAppStateManager.NormalExitAction

This enumeration governs when the NormalExit method suspends a task by calling WaitForCarbonUnit.

Namespace: WizardWrx.ConsoleAppAids3
Assembly: WizardWrx.ConsoleAppAids3.dll
Syntax
public enum NormalExitAction

Fields

Name Description
ExitImmediately

Always exit immediately, regardless of the status code.
HaltOnError

Exit immediately if the status code is ERROR_SUCCESS (zero). Otherwise, call WaitForCarbonUnit.
Silent

Exit silently, without displaying any text on the console.
Timed

Exit after allowing a specified amount of time for an operator to read the error message.
WaitForOperator

Always call WaitForCarbonUnit.
